#### afficher un code sur plusieurs lignes avec trois apostrophes inversées
```
{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}
```

pour que lle bloc de code soit colorisé selon le langage, spécifier le langage apres les trois premiere apostrophes  
ici bloc python
```python
nb = 7
i = 0

while i < 10:
    print(i + 1, "*", nb, "=", (i + 1) * nb)
    i += 1
```
